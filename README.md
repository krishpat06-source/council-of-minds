# Council of Minds

Council of Minds is an AI-powered podcast and debate platform where history's greatest thinkers discuss modern questions.

## Overview

Instead of receiving a single chatbot answer, users experience a roundtable discussion between legendary historical figures such as:

* Albert Einstein
* Nikola Tesla
* Leonardo da Vinci
* Alan Turing
* Marie Curie

Each figure provides unique perspectives, creating an engaging and educational debate experience.

## Features

* Premium dark-themed UI
* Historical figure profiles
* Podcast-style discussion experience
* Interactive debate interface
* Responsive modern design

## Tech Stack

* Vite
* JavaScript
* HTML
* CSS

## Vision

To combine artificial intelligence, education, and historical wisdom into a single immersive platform where users can learn from the greatest minds in history.
